# 덱스컴 알람 (Dexcom Alarm)

자녀의 덱스컴 센서 교체 시기를 관리하기 위한 웹 앱(PWA)입니다. 두 명의 자녀를 개별적으로 관리할 수 있으며, 10일의 기본 사용 기간과 12시간의 유예 기간을 고려하여 남은 시간을 직관적으로 보여줍니다.

## 주요 기능

- **듀얼 대시보드**: 두 자녀의 센서 상태를 한 눈에 확인
- **정밀한 타이머**: 초 단위 실시간 카운트다운 (D-day 및 남은 시간 표시)
- **스마트 알림**: 종료 1일 전 또는 사용자 설정 시간에 맞춰 브라우저 알림 발송
- **수동 시간 수정**: 센서 시작 시간을 직접 수정하여 정확한 일정 관리 가능
- **PWA 지원**: 모바일 기기 홈 화면에 추가하여 앱처럼 사용 가능
- **데이터 로컬 저장**: 별도의 가입 없이 브라우저 내에 안전하게 데이터 보관

## 설치 및 실행 방법

### 웹에서 바로 사용하기
- [깃허브 페이지 주소](링크_예정)에 접속하여 사용합니다.

### 모바일 앱으로 설치 (PWA)
1. 브라우저(Safari, Chrome 등)로 해당 주소에 접속합니다.
2. 브라우저의 '공유' 또는 '설정' 메뉴에서 **'홈 화면에 추가'**를 선택합니다.
3. 홈 화면에 생성된 아이콘을 통해 앱처럼 간편하게 실행할 수 있습니다.

## 기술 스택

- **PWA**: Service Worker, Web App Manifest
